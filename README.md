# build-mysql-indexes
Script to add an index to a column all tables in a MySQL database.

````
./build-mysql-indexes --help
Usage of ./build-mysql-indexes:
  -D="mysql": Database
  -P=3306: Port
  -column="updated": Column to be indexed
  -h="localhost": Hostname
  -p="pass": Password
  -u="root": Username
````
